# Big Tex Pool Supplies Starter Repo

A clean Next.js starter site for the Big Tex Pool Supplies rebuild.

## Purpose

This repo is designed as a staging replacement for the current GoDaddy-style site. Once access to the existing web account/assets is available, content and live ordering links can be ported into this codebase.

## Current positioning

Big Tex Pool Supplies is framed as a Houston pool supply, commercial delivery, and hard-to-find parts partner.

Primary customer paths:

1. Retail and online ordering
2. Commercial supply support
3. Hard-to-find pool parts
4. Future managed inventory, alert intake, and logistics support

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000.
